# Python Fundamentals Lab (Sessions 1–2)

## Setup Instructions

### Activate the Virtual Environment

**On Windows:**
```bash
.venv\Scripts\activate
```

**On macOS/Linux:**
```bash
source .venv/bin/activate
```

### Running Each Exercise

Once the virtual environment is activated, run each exercise using:

```bash
python ex1_intro.py          # Exercise 1: I/O + types
python ex2_conditionals.py   # Exercise 2: Conditionals
python ex3_for_loop_stats.py # Exercise 3: For loop + statistics
python ex4_validation.py     # Exercise 4: While loop + validation
python ex5_menu_conversion.py # Exercise 5: Menu program
python ex6_morpion.py        # Exercise 6: Tic-Tac-Toe game
```

- **Ex1**: Basic CLI interaction (input/output, type conversion)
- **Ex2**: Decision making with if/elif/else
